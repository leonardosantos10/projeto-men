o que é HTML? ferramentas Utilizadas?

(voltar)

o que é HTML (HyperText Markup Language) é uma linguagem de marcação de Hipertexto utilizada para estruturar e organizar conteúdo na web.
Ele define a estrutura básica de páginas web, como cabeçalhos, parágrafos, listas, links, imagens e outros elementos visuais e interativos.

Para escrever e executar códigos HTML de forma eficiente, é recomendado usar o Visual Studio Code (VSCode),
um editor de texto poderoso com funcionalidades como autocompletar e realce de sintaxe.

O VSCode também permite a instalação de Plugins como o Live Server, que cria um servidor local na sua máquina.

Com o Live Server, você pode ver o resultado das alterações no seu código HTML em tempo real no navegador, sem precisar atualizar a página manualmente.
Essa combinação de VSCode e Live Server torna o processo de desenvolvimento web mais rápido e prático.

inspetor de elementos

(voltar)

O Inspetor de Elementos é uma ferramenta presente em navegadores como o Google Chrome e outros, como Firefox e Edge,
permitindo aos desenvolvedores inspecionar e alterar temporariamente o código HTML, CSS de uma página web

A ferramenta, acessível pelo atalho F12 ou pelo menu no canto superior lado direito nos três pontinhos > Mais ferramentas > Ferramentas do desenvolvedor, possibilita testar mudanças em tempo real, sem modificar o código original.

Através dela, você pode visualizar a estrutura da página e testar novos recursos de forma imediata.

Essa ferramenta é essencial para testes rápidos e otimização do desenvolvimento de sites, ajudando a entender como o código funciona e a identificar problemas.


Estrutura básica do HTML

(voltar)

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre será dessa forma e você sempre, sempre começará seu HTML começando por esse código.

Geralmente os editores como o Sublime Text já tem atalhos para iniciar os documentos HTMLs com essa estrutura, logo, você não precisa se preocupar em decorá-la, mas é bom que faça. Veja abaixo como ela se inicia:

  1. <!DOCTYPE html>
  2. <html lang="pt-br">
  3.   <head>
  4.     <title>Título da página</title>
  5.     <meta charset="utf-8">
  6.   </head>
  7.   <body>
  8.     Aqui vai o código HTML que fará seu site aparecer.
  9.   </body>
  10. </html>

É possível compreender o documento em HTML de uma maneira muito simples, através de uma divisão de blocos das tags essenciais, conforme a a seguinte estrutura:

  • Definição do documento (doctype)
  • Cabeça (head)
  • Corpo (body)

Doctype - Definindo o documento

Uma coisa importante: SEMPRE deve existir o doctype, que é este código .
O doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site, que o código encontrado ali é um código HTML.

HEAD

Contém informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do documento: vinculação com outros arquivos,

BODY

é o corpo do site o código HTML que fará seu site aparecer.Dentro do elemento BODY sua estrutura de página terá os elementos semânticos da construção da sua página,
onde serão declarados e identificados cabeçalhos, rodapé, conteúdo principal, etc.

Atributos das tag

(voltar)

As tags HTML possuem atributos que definem propriedades de seus elementos. Alguns atributos são mais globais e podem ser usados em diversas tags, como o atributo id, que serve para identificar unicamente um elemento na página.

Outros atributos são mais específicos, como o style, usado para aplicar estilos CSS diretamente na tag.

o atributo type na tag 'input' que define o tipo de dado que o formulário aceitará (como texto ou número).

A tag "img" usa o atributo "src" para definir a fonte da imagem e pode ajustar a largura e altura com os atributos width e height

Além disso, o JavaScript pode interagir com os elementos HTML, alterando seu conteúdo usando identificadores como o "id"
